# Sip Hook
React Hook para trabalhar com SipJS

## Dependência:
- [SipJs](https://sipjs.com/)

## Funcionalidades:

 - Realizar Chamada ✅
 - Atende Chamada ✅
 - Desligar Chamada ✅
 - Recusar Chamada ✅
 - Identificação de Chamada ✅
- Chamada em Espera ❌
- Remover Chamada de Espera ❌
- Chamada em Mudo ❌
- Remover chamada do Mudo ❌
- Envio DTMF ✅
- Auto Atendimento ✅
- Transferência 🚧
